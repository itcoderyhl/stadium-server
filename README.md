# stadium-server
基于SpringBoot+Vue前后端分离的羽毛球馆预约系统（源码+文档），系统包含管理员、用户角色，功能如下： 管理员：首页统计；用户管理；场地管理；教练管理；课程管理；器材商城（商品种类、商品管理、订单管理、评论管理）；比赛管理；运动知识；系统留言；系统公告；个人信息；密码修改。 用户：注册、登录系统；首页；场地信息（分类、场地列表、场地详情、预约）；教练信息（等级、教练列表、详情、点赞、预约）；课程信息（课程列表、收藏、解锁）；器材商城（商品分类、商品列表、商品详情、加入购物车、下单、评论）；比赛信息；运动技巧；系统公告。用户还具有个人中心功能，功能包括：场地预约；教练预约；教练评价；教练点赞；课程解锁；课程评价；课程收藏；我的订单；商品评论；系统留言；个人资料；修改密码。
