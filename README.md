## 【原创】springboot+vue前后端分离的羽毛球馆预约系统（源码+文档）

## 系统介绍

基于SpringBoot+Vue前后端分离的羽毛球馆预约系统（源码+文档），系统包含管理员、用户角色，功能如下：
管理员：首页统计（场地统计、7天订单金额统计）；用户管理；场地管理（场地分类、场地管理、场地预约）；教练管理（教练管理、预约管理、评价管理）；课程管理（课程管理、课程解锁、课程评价）；器材商城（商品种类、商品管理、订单管理、评论管理）；比赛管理；运动知识；系统留言；系统公告；个人信息；密码修改。
用户：注册、登录系统；首页；场地信息（分类、场地列表、场地详情、预约）；教练信息（等级、教练列表、详情、点赞、预约）；课程信息（课程列表、收藏、解锁）；器材商城（商品分类、商品列表、商品详情、加入购物车、下单、评论）；比赛信息；运动技巧；系统公告。用户还具有个人中心功能，功能包括：场地预约；教练预约；教练评价；教练点赞；课程解锁；课程评价；课程收藏；我的订单；商品评论；系统留言；个人资料；修改密码。
系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，适合作为毕业设计、课程设计，有需要可提供指导。

项目由本人原创开发，经过多次测试，项目运行无问题，请放心使用，作者QQ：1344127185。

## 使用技术

服务器端技术栈：

- Springboot
- SpringMvc
- Mybatis
- MySQL

前端技术栈：

- vue
- elementUI
- axios
- echarts

## 环境介绍

- IDEA集成环境
- JDK1.8
- MySQL5.6+

可提供所有运行软件、远程安装、技术指导。
